# Cervical Cancer Awareness – Interactive Quiz

An interactive, single-page web application designed for presentations and workshops to raise awareness about cervical cancer, its prevention, and screening methods.

## Features

* **Interactive Slides:** Smooth, animated navigation through a series of engaging slides.
* **Icebreaker & Discussion:** Interactive bubbles to gauge the audience's prior knowledge and facilitate group discussions.
* **Myth vs. Fact:** Clickable cards to instantly debunk common misconceptions.
* **Multiple Choice Quiz:** 4-question quiz with real-time scoring and correct-answer highlighting.
* **Rapid Fire Round:** Quick true/false questions for fast-paced learning and engagement.
* **Role Play Activity:** Built-in scenarios to help participants build confidence in discussing medical screening.
* **Feedback Collection:** Interactive text areas to capture participant takeaways and action items.

## How to Run

1. Download or clone this repository to your local machine.
2. Double-click `index.html` to open it in any modern web browser (Chrome, Firefox, Safari, Edge).
3. Use the on-screen **Next** and **Back** buttons to navigate.
4. **Keyboard Shortcuts:** 
   - Use the `Left` and `Right` arrow keys (or `Spacebar`) to switch slides.
   - Press `F` to toggle Fullscreen mode for a better presentation experience.
